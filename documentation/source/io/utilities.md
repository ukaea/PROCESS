
# Utilites

There are a number of utilities available with `PROCESS`. These are accessible in various modules in the io package or through the `PROCESS` CLI.

To see available options please see the following sub command and their help documentation. All subcommand help is available using `process <subcommand> --help`, help is also available for each constituent command.

`indat`:
    Creation and modification of input files

`mfile`:
    Tools for comparison and conversion of MFILEs

`plot`:
    Tools for plotting various set of `PROCESS` output
